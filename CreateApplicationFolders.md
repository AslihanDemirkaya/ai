# Create Application Folders Prompt

## Objective
Generate application folders and documents for job listings using the candidate's supporting materials.

## Steps
1. Read the file: `job_list_summary.md`.
2. Create a folder named `ApplicationFolder`.
3. Inside `ApplicationFolder`, create subfolders for each listing using the company or university name.
4. In each subfolder, create the required application documents listed for that job.
   - Each document filename should use the candidate's last name followed by an underscore and the required document type.
   - Example: `Demirkaya_CoverLetter`
5. Use supporting documents to create the files:
   - `CV/` contains the candidate's CV.
   - `Sample_Cover_Letter/` contains a sample cover letter.
6. When creating a new cover letter, update it to include the specific university or company name for that job.

## Notes
- The generated documents should reflect the candidate's qualifications and the target employer.
- Keep filenames consistent and descriptive.
- Ensure each application folder includes all required documents for that position.
