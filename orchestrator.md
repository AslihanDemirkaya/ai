# Orchestrator Prompt

## Objective
Guide the workflow for finding jobs and creating application folders.

## Steps
1. Ask the user/candidate for their name.
2. Run `JobSearch.md`.
3. Run `CreateApplicationFolders.md`.

## Notes
- Ensure the candidate's name is captured before proceeding.
- Use the output of `JobSearch.md` to drive `CreateApplicationFolders.md`.
